# Button_Server2
안드로이드 어플 'Button'의 메인 서버입니다.    
This is the main server for an android application called "Button"   
_______________      
#### 프로젝트 기능 
   1. 로그인 및 회원가입 
   2. 아이디 및 비밀번호 찾기 
   3. 내 프로필 페이지 (내 정보 수정 / 비밀번호 변경 / 계정 탈퇴)
   4. 내 옷 등록 / 수정 / 삭제 
   5. 직접 코디 등록 / 수정 / 삭제 
   6. 친구추가 및 친구 위해 코디하기 
   7. 달력에 코디 등록 / 수정 / 삭제
