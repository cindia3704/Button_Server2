# Button_Server2
안드로이드 어플 'Button'의 메인 서버입니다.    
This is the main server for an android application called "Button"   
해당 서버는 Django Rest Framework를 이용하여 구현되었습니다. 
_______________      
#### 프로젝트 기능: 
   1. 로그인 및 회원가입 
   2. 아이디 및 비밀번호 찾기 
   3. 내 프로필 페이지 (내 정보 수정 / 비밀번호 변경 / 계정 탈퇴)
   4. 내 옷 등록 / 수정 / 삭제 
   5. 직접 코디 등록 / 수정 / 삭제 
   6. 친구 추가 / 삭제 
   7. 친구 위해 코디 등록 / 수정 / 삭제 
   8. 달력에 코디 등록 / 수정 / 삭제 
   9. 상황입력 & 계절에 따른 코디 추천
